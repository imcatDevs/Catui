빠른 시작

Step 1 — 다운로드
downloadIMCAT UI v1.1.2 다운로드 imcatui-1.1.2.zip (약 400KB) — JS + CSS + 폰트 포함
또는 npm으로 설치: npm install @imcat-ckim/catui
Step 2 — HTML에 추가
압축 해제 후 dist/ 폴더를 프로젝트에 복사하세요.
<!DOCTYPE html>
<html lang="ko">
<head>
  <!-- CSS 로드 -->
  <link rel="stylesheet" href="dist/imcat-ui.css">
</head>
<body>

  <h1>Hello IMCAT UI</h1>
  <button class="btn btn--primary" onclick="run()">클릭</button>

  <!-- JS 로드 (body 끝) -->
  <script src="dist/imcat-ui.min.js"></script>
</body>
</html>
Step 3 — 사용하기 
// 단축 API — 모달, 토스트, 확인 다이얼로그
await IMCAT.alert('Hello!');
if (await IMCAT.confirm('계속?')) {
  IMCAT.toast.success('완료!');
}

// 모듈 동적 로드 — 필요한 모듈만 자동으로 가져옵니다
const { Modal } = await IMCAT.use('overlays');
new Modal({ title: '제목', content: '내용' }).show();

// DOM 조작 — jQuery 스타일 체이닝
IMCAT('#app').addClass('active').text('Ready!');
