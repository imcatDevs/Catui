Forms

FileUpload, Rating, SignaturePad, FormWizard 컴포넌트

FileUpload — 기본
드래그 앤 드롭 또는 클릭으로 파일을 업로드합니다. 이미지 미리보기, 파일 크기/타입 검증 포함.
const { FileUpload } = await IMCAT.use('forms');
new FileUpload('#container', {
  accept: 'image/*,.pdf',
  maxSize: 5 * 1024 * 1024,  // 5MB
  multiple: true,
  maxFiles: 10,
  dropzone: true,
  preview: true
});
FileUpload — 단일 파일 & 진행률
단일 파일 모드 + showProgress 옵션으로 업로드 시뮬레이션.

파일을 선택한 후 버튼을 클릭하세요.

const fu = new FileUpload('#container', {
  accept: 'image/*',
  multiple: false,
  showProgress: true,
  maxSize: 2 * 1024 * 1024,
  onChange: (files) => console.log(files)
});
// 업로드 시뮬레이션 (2초)
await fu.simulateUpload(2000);
Rating — 기본
별점 입력. 클릭으로 값 설정, 마우스 오버 시 하이라이트.
기본: 3점
비활성: (readonly) 
const { Rating } = await IMCAT.use('forms');
new Rating('#container', {
  max: 5,
  value: 3,
  onChange: (val) => console.log(val)
});
// 읽기 전용
new Rating('#readonly', { value: 4, readonly: true });
Rating — 크기 & 커스텀
size 옵션(sm/md/lg), 커스텀 아이콘, 색상 변경.

// 크기 변형
new Rating('#sm', { size: 'sm', value: 2 });
new Rating('#lg', { size: 'lg', value: 4 });
// 하트 아이콘 + 빨간색
new Rating('#heart', {
  icon: 'favorite',
  iconEmpty: 'favorite_border',
  color: '#ef4444',
  value: 3
});
// 10점 척도
new Rating('#ten', { max: 10, value: 7, size: 'sm' });
SignaturePad — 기본
마우스/터치로 서명을 그립니다. 지우기/저장 버튼 내장, base64 이미지 추출.
const { SignaturePad } = await IMCAT.use('forms');
const pad = new SignaturePad('#container', {
  width: 400,
  height: 200,
  penColor: '#000',
  penWidth: 2,
  backgroundColor: '#fff'
});
pad.toDataURL();    // base64 PNG
pad.isEmpty();       // 비어있는지 확인
pad.clear();
SignaturePad — 커스텀 스타일
펜 색상, 두께, 배경색을 변경합니다.

빨간 펜 (크림 배경)

FormWizard — 기본
단계별 폼. 프로그레스 바, 이전/다음 네비게이션, 유효성 검사 지원.
const { FormWizard } = await IMCAT.use('forms');
new FormWizard('#container', {
  steps: [
    { title: '기본 정보', content: '<form>...</form>' },
    { title: '상세 정보', content: '...' },
    { title: '완료', content: '...' }
  ],
  onStepChange: (step) => console.log(step),
  onComplete: (data) => console.log(data)
});
FormWizard — 유효성 검사 & 데이터 수집
validate 함수로 단계별 유효성 검사. onComplete에서 모든 폼 데이터 자동 수집.

new FormWizard('#container', {
  steps: [
    {
      title: '계정',
      content: '<input name="email" ...>',
      validate: async () => {
        const v = document.querySelector('[name="email"]').value;
        return v.includes('@');
      }
    },
    ...
  ],
  onComplete: (data) => {
    console.log('수집된 데이터:', data);
  }
});
