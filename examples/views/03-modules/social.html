Social

SocialFeed, ChatUI, Comments, ShareButtons, Reactions 컴포넌트

SocialFeed

포스트 작성, 좋아요, 댓글, 공유 기능을 제공합니다

const { SocialFeed } = await IMCAT.use('social');

new SocialFeed('#container', {
  currentUser: { id: 'me', name: '나', avatar: '' },
  showCompose: true,
  posts: [
    {
      id: '1',
      userName: '홍길동',
      text: '프로젝트가 완성되었습니다!',
      createdAt: '2025-03-14T14:30:00',
      likes: 12,
      commentCount: 3
    }
  ],
  onPost: (post) => console.log('게시:', post),
  onLike: (id) => console.log('좋아요:', id)
});
ChatUI

실시간 채팅 인터페이스 — 메시지 그룹화, 날짜 구분, 타이핑 인디케이터

const { ChatUI } = await IMCAT.use('social');

const chat = new ChatUI('#container', {
  currentUser: { id: 'me', name: '나' },
  messages: [
    { userId: 'hong', userName: '홍길동', text: '안녕하세요!',
      timestamp: '2025-03-14T14:30:00' },
    { userId: 'me', userName: '나', text: '반갑습니다!',
      timestamp: '2025-03-14T14:31:00' }
  ],
  showTimestamp: true,
  groupMessages: true,
  onSend: (msg) => console.log(msg)
});

// 타이핑 인디케이터
chat.showTyping([{ name: '홍길동' }]);

// 메시지 추가
chat.addMessage({ userId: 'hong', userName: '홍길동',
  text: '새 메시지', timestamp: new Date().toISOString() });
Comments

댓글/답글 시스템 — 좋아요, 수정, 삭제, 중첩 답글 지원

const { Comments } = await IMCAT.use('social');

new Comments('#container', {
  currentUser: { id: 'me', name: '나' },
  comments: [
    {
      id: 'c1', userId: 'hong', userName: '홍길동',
      text: '좋은 글입니다!',
      createdAt: '2025-03-14T10:00:00',
      likes: 5,
      replies: [
        { id: 'r1', userId: 'kim', userName: '김영희',
          text: '동의합니다!', createdAt: '2025-03-14T11:00:00' }
      ]
    }
  ],
  allowReplies: true,
  maxDepth: 3,
  onSubmit: (c) => console.log('댓글:', c)
});
ShareButtons

new ShareButtons('#container', {
  url: 'https://imcat.dev',
  title: 'IMCAT UI Framework',
  platforms: ['copy', 'twitter', 'facebook', 'linkedin', 'email'],

Reactions

이모지 리액션 — 토글, 카운터, 커스텀 이모지

기본

커스텀 이모지

const { Reactions } = await IMCAT.use('social');

new Reactions('#container', {
  reactions: [
    { emoji: '👍', label: '좋아요', count: 12 },
    { emoji: '❤️', label: '사랑해요', count: 8 },
    { emoji: '😂', label: '웃겨요', count: 3 }
  ],
  onReact: (emoji, active) => console.log(emoji, active)
