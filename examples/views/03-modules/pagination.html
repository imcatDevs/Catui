Pagination

페이지네이션 컴포넌트 — 다양한 스타일과 사이즈 변형 지원

Default Pagination

기본 페이지네이션. 처음/마지막/이전/다음 버튼 + 생략 부호 포함.

const { Pagination } = await IMCAT.use('pagination');
new Pagination('#container', {
  totalItems: 200,
  itemsPerPage: 10,
  currentPage: 1,
  showInfo: true,
  onChange: (pageData) => console.log(pageData)
});
Rounded Pagination

rounded 스타일. 원형 버튼.

new Pagination('#container', {
  totalItems: 150,
  rounded: true,   // 원형 스타일
  currentPage: 3
});
Sizes (Small / Large)

size: 'sm' / size: 'lg' 사이즈 변형.

Large

Small

Small + Rounded

// Large
new Pagination('#lg', { size: 'lg', totalItems: 80 });
// Small
new Pagination('#sm', { size: 'sm', totalItems: 80 });
// Small + Rounded
new Pagination('#sm-r', { size: 'sm', rounded: true, totalItems: 80 });
정렬 + Info 텍스트

중앙 정렬 + 정보

우측 정렬

텍스트 라벨 (아이콘 OFF)

new Pagination('#center', {
  align: 'center',
  showInfo: true,
  infoTemplate: '{{currentPage}} / {{totalPages}} 페이지'
});
new Pagination('#end', { align: 'end' });
new Pagination('#text', { icons: false });
DataPaginator (테이블 + 검색)

JSON 데이터 기반 페이지네이션. 검색, 정렬, 페이지 크기 변경 지원.

const { DataPaginator } = await IMCAT.use('pagination');
const dp = new DataPaginator({
  container: '#container',
  data: users,
  itemsPerPage: 10,
  renderItem: (item) => '...',
  paginationOptions: { showInfo: true, rounded: true }
});
dp.search('검색어', ['name', 'email']);
dp.sort((a, b) => a.name.localeCompare(b.name));
