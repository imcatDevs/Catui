Feedback

Toast, Notification, ProgressTracker, Skeleton 컴포넌트

Toast
IMCAT.toast.success('저장 완료!');
IMCAT.toast.error('오류 발생');
IMCAT.toast.warning('주의');
IMCAT.toast.info('안내', 5000);
Notification
const { Notification } = await IMCAT.use('feedback');
Notification.show({
  title: '새 메시지',
  message: '홍길동님이 메시지를 보냈습니다.',
  type: 'info',
  duration: 5000
});
ProgressTracker
const { ProgressTracker } = await IMCAT.use('feedback');
const pt = new ProgressTracker('#container', {
  steps: ['계정 생성', '프로필 설정', '결제 정보', '완료'],
  current: 1,
  clickable: true
});
pt.next();  // 다음 단계
pt.prev();  // 이전 단계
Skeleton — 카드 비교
실제 카드와 스켈레톤 로딩 카드를 비교합니다. 크기와 비율이 동일합니다.
image
카드 제목

간단한 설명 텍스트입니다. 카드 내용을 채웁니다.

<!-- 스켈레톤 카드 -->
<div class="card" aria-hidden="true">
  <div class="skeleton skeleton--animated" style="height:140px"></div>
  <div class="card__body placeholder-glow">
    <div class="skeleton skeleton--title"></div>
    <div class="skeleton skeleton--text"></div>
  </div>
</div>
Skeleton — 작동 원리
.skeleton 클래스로 로딩 플레이스홀더를 만듭니다. 너비는 인라인 스타일이나 유틸리티로 지정합니다.

             
<span class="skeleton skeleton--text skeleton--animated" style="width:50%"></span>

<!-- 버튼 플레이스홀더 -->
<a class="btn btn--primary" style="pointer-events:none;opacity:0.65">
  <span class="skeleton skeleton--animated" style="position:absolute;inset:0"></span>
</a>
Skeleton — 너비
인라인 스타일로 너비를 자유롭게 조절할 수 있습니다.
<span class="skeleton skeleton--text skeleton--animated" style="width:100%"></span>
<span class="skeleton skeleton--text skeleton--animated" style="width:75%"></span>
<span class="skeleton skeleton--text skeleton--animated" style="width:50%"></span>
<span class="skeleton skeleton--text skeleton--animated" style="width:25%"></span>
Skeleton — 컬러
.skeleton--{color} 수식어로 컬러를 변경할 수 있습니다.
<span class="skeleton skeleton--text"></span>              <!-- 기본 -->
<span class="skeleton skeleton--text skeleton--primary"></span>
<span class="skeleton skeleton--text skeleton--success"></span>
<span class="skeleton skeleton--text skeleton--danger"></span>
Skeleton — 크기
.skeleton--lg, .skeleton--sm, .skeleton--xs로 높이를 조절합니다.

Large (.skeleton--lg)

기본 (.skeleton--text)

Small (.skeleton--sm)

Extra Small (.skeleton--xs)

<span class="skeleton skeleton--lg skeleton--animated"></span>
<span class="skeleton skeleton--text skeleton--animated"></span>
<span class="skeleton skeleton--sm skeleton--animated"></span>
<span class="skeleton skeleton--xs skeleton--animated"></span>
Skeleton — 애니메이션
.placeholder-glow (글로우) 또는 .placeholder-wave (웨이브) 래퍼로 애니메이션을 적용합니다.

Glow (깜빡임)

Wave (흐름)

Pulse (맥박)

<!-- Glow: 부모 래퍼 방식 -->
<div class="placeholder-glow">
  <span class="skeleton skeleton--text"></span>
</div>

<!-- Wave: 부모 래퍼 방식 -->
<div class="placeholder-wave">
  <span class="skeleton skeleton--text"></span>
</div>

<!-- Pulse: 개별 클래스 방식 -->
<span class="skeleton skeleton--text skeleton--pulse"></span>
Skeleton — JS API
JavaScript Skeleton 클래스로 동적 플레이스홀더를 생성합니다.

type: 'text'

type: 'avatar'

type: 'card'

type: 'list'

const { Skeleton } = await IMCAT.use('feedback');

// 텍스트 스켈레톤
new Skeleton('#container', { type: 'text', lines: 3 });

// 아바타 스켈레톤
new Skeleton('#container', { type: 'avatar' });

// 카드 스켈레톤
new Skeleton('#container', { type: 'card' });

// 리스트 스켈레톤
new Skeleton('#container', { type: 'list', lines: 3 });
