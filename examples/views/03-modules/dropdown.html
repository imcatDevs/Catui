Dropdown

드롭다운 메뉴와 컨텍스트 메뉴 컴포넌트

기본 Dropdown
const Dropdown = await IMCAT.use('dropdown');
new Dropdown('#trigger', {
  items: [
    { text: '편집', icon: 'edit', action: () => {} },
    { text: '삭제', icon: 'delete', danger: true },
    { divider: true },
    { text: '설정', icon: 'settings' }
  ]
});
버튼 색상 변형
모든 버튼 스타일에 드롭다운을 적용할 수 있습니다.
// 모든 btn 변형에 적용 가능
new Dropdown('#btn-primary', { items: [...] });
new Dropdown('#btn-danger', { items: [...] });
방향 변형
position 옵션으로 메뉴가 열리는 방향을 지정할 수 있습니다.
new Dropdown('#trigger', {
  items: [...],
  position: 'top'      // 'top' | 'bottom' | 'left' | 'right'
});
Hover 트리거
openOnHover 옵션으로 마우스 오버 시 자동으로 메뉴를 엽니다.
new Dropdown('#trigger', {
  items: [...],
  openOnHover: true,
  hoverDelay: 200   // ms
});
활성화 & 비활성화 아이템
아이템에 active, disabled 속성을 지정할 수 있습니다.
new Dropdown('#trigger', {
  items: [
    { text: '일반 항목' },
    { text: '활성 항목', active: true },
    { text: '비활성 항목', disabled: true },
    { divider: true },
    { text: '위험 항목', danger: true }
  ]
});
헤더 구분
header 속성으로 메뉴 섹션을 구분할 수 있습니다.
new Dropdown('#trigger', {
  items: [
    { header: '파일' },
    { text: '새 파일', icon: 'add' },
    { text: '열기', icon: 'folder_open' },
    { header: '편집' },
    { text: '복사', icon: 'content_copy' },
    { text: '잘라내기', icon: 'content_cut' }
  ]
});
메뉴 정렬
align 옵션으로 메뉴의 정렬 위치를 조정할 수 있습니다.
new Dropdown('#trigger', {
  items: [...],
  align: 'end'    // 'start' | 'center' | 'end'
});
아이콘 트리거
아이콘 버튼으로 드롭다운을 트리거할 수 있습니다.
// 아이콘 버튼에 적용
new Dropdown('#icon-btn', {
  items: [...],
  align: 'end'
});
스크롤 가능 드롭다운
항목이 많은 경우 자동으로 스크롤이 적용됩니다.
// 아이템이 많으면 자동 스크롤
new Dropdown('#trigger', {
  items: Array.from({ length: 15 }, (_, i) =>
    ({ text: `항목 ${i+1}`, icon: 'circle' })
  )
});
Context Menu (우클릭)
contextmenu 이벤트와 조합하여 우클릭 메뉴를 만들 수 있습니다.
이 영역에서 우클릭하세요
var dd = new Dropdown('#area', {
  items: [
    { text: '복사', icon: 'content_copy' },
    { text: '붙여넣기', icon: 'content_paste' },
    { divider: true },
    { text: '삭제', icon: 'delete', danger: true }
  ]
});
el.addEventListener('contextmenu', e => {
  e.preventDefault(); dd.show();
});
