Overlays

Modal, Drawer, Offcanvas — 오버레이 UI 컴포넌트

Modal — 크기 변형
size 옵션으로 모달 크기를 조절합니다. sm / md / lg / xl 4단계.
const { Modal } = await IMCAT.use('overlays');
new Modal({
  title: '모달 제목',
  content: '<p>모달 내용</p>',
  size: 'md'  // 'sm' | 'md' | 'lg' | 'xl'
}).show();
Modal — 버튼 & 액션
buttons 배열로 푸터 버튼을 추가합니다. close: true로 자동 닫기, action 함수로 커스텀 동작.
new Modal({
  title: '작업 확인',
  content: '<p>이 작업을 진행하시겠습니까?</p>',
  buttons: [
    { text: '취소', variant: 'secondary', close: true },
    { text: '확인', variant: 'primary', close: true,
      action: () => console.log('확인됨') }
  ]
}).show();
Modal — 옵션 변형
centered, scrollable, fullscreen, 백드롭/키보드 옵션 등.
// 수직 중앙 정렬
new Modal({ centered: true, ... });
// 스크롤 가능한 긴 콘텐츠
new Modal({ scrollable: true, ... });
// 풀스크린
new Modal({ fullscreen: true, ... });
// 백드롭 클릭으로 닫기 비활성
new Modal({ backdropClose: false, ... });
단축 API (alert / confirm / prompt)
Modal.alert(), Modal.confirm() 정적 메서드 및 IMCAT.alert/confirm 단축키.

// 알림 — 확인 버튼만
await IMCAT.alert('저장이 완료되었습니다.');
// 확인 — true/false 반환
const ok = await IMCAT.confirm('삭제하시겠습니까?');
// 입력 — 문자열 또는 null
const name = await IMCAT.prompt('이름을 입력하세요');
Drawer — 방향
position 옵션으로 4방향 슬라이드 패널. left / right / top / bottom.
const { Drawer } = await IMCAT.use('overlays');
new Drawer({
  title: '설정',
  content: '<p>패널 내용</p>',
  position: 'right',  // 'left' | 'right' | 'top' | 'bottom'
  width: '360px'
}).show();
Drawer — 리치 콘텐츠
Drawer 내부에 메뉴, 폼, 설정 패널 등 다양한 콘텐츠를 배치할 수 있습니다.
Offcanvas
Drawer의 변형. 동일 API에 offcanvas 클래스가 추가 적용됩니다.
const { Offcanvas } = await IMCAT.use('overlays');
new Offcanvas({
  title: '사이드 패널',
  content: '<p>Offcanvas 내용</p>',
  position: 'right'
}).show();
이벤트 시스템
모든 오버레이는 EventEmitterMixin 기반 이벤트를 지원합니다.
const modal = new Modal({ title: '이벤트 테스트', ... });
modal.on('beforeShow', () => console.log('열기 전'));
modal.on('show', () => console.log('열림'));
modal.on('beforeHide', () => console.log('닫기 전'));
modal.on('hide', () => console.log('닫힘'));
modal.show();
// destroy() — 모든 이벤트 해제 + DOM 제거
