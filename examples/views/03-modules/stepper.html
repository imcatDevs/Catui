Stepper

단계별 진행 표시 컴포넌트 — Form Wizard, 프로세스 흐름 등에 활용

Basic Wizard

아이콘 + 폼 콘텐츠 + 이전/다음 버튼 구성의 기본 마법사 패턴입니다.

const { Stepper } = await IMCAT.use('stepper');
const wizard = new Stepper('#container', {
  steps: [
    { title: '계정', icon: 'person', content: '<form>...</form>' },
    { title: '프로필', icon: 'badge', content: '...' },
    { title: '완료', icon: 'check_circle', content: '...' }
  ]
});
wizard.next();   // 다음 단계 (현재 단계 완료 처리 후 이동)
wizard.prev();   // 이전 단계
wizard.goTo(2);  // 특정 단계로 이동
wizard.reset();  // 초기화

Vertical Stepper

각 단계에 콘텐츠가 인라인 표시. 이전/다음 버튼, 편집 기능 포함.

const { VerticalStepper } = await IMCAT.use('stepper');
new VerticalStepper('#container', {
  steps: [
    { title: '계정 설정', subtitle: '사용자 정보', content: '<form>...</form>' },
    { title: '프로필', subtitle: '상세 정보', content: '...' },
    { title: '완료', subtitle: '확인', content: '...' }
  ],
  editable: true,     // 완료 단계 수정 가능
  expandable: true   // 클릭으로 펼치기/접기
});
Linear 모드 (순차 진행)

linear: true — 이전 단계를 완료해야만 다음 단계로 이동 가능.

new Stepper('#container', {
  linear: true,
  clickable: true,
  onComplete: () => alert('모든 단계 완료!')
});
