Carousel

이미지/콘텐츠 슬라이더 컴포넌트 — 다양한 전환 효과, 캡션, 멀티 슬라이드, 썸네일, 반응형 지원

Slides Only
화살표, 도트 없이 자동 재생만 동작하는 가장 단순한 형태입니다.
new Carousel('#container', {
  items: [...],
  autoplay: true,
  autoplaySpeed: 3000,
  arrows: false,
  dots: false,
  draggable: false
});
With Controls
좌우 화살표(arrows: true)를 추가하여 수동으로 슬라이드를 전환합니다.
new Carousel('#container', {
  items: [...],
  arrows: true,
  dots: false
});
With Indicators
하단 도트 인디케이터(dots: true)와 화살표를 함께 표시합니다.
new Carousel('#container', {
  items: [...],
  autoplay: true,
  autoplaySpeed: 3000,
  dots: true,
  arrows: true
});
With Captions
슬라이드 아이템에 title과 description을 지정하면 이미지 위에 캡션 오버레이가 표시됩니다.
new Carousel('#container', {
  items: [
    { src: 'img.jpg', title: '제목', description: '설명 텍스트' },
    ...
  ],
  arrows: true,
  dots: true
});
Crossfade
effect: 'fade'로 슬라이드 대신 페이드 전환을 적용합니다.
new Carousel('#container', {
  items: [...],
  effect: 'fade',
  autoplay: true
});
전환 효과 모음
IMCAT Carousel은 5가지 전환 효과를 지원합니다: slide, fade, scale, flip, cube

Scale 효과

Flip 효과

// 사용 가능한 effect 값
effect: 'slide'  // 기본 수평 슬라이드
effect: 'fade'   // 페이드 인/아웃
effect: 'scale'  // 축소/확대 전환
effect: 'flip'   // Y축 회전 전환
effect: 'cube'   // 큐브 회전 전환
멀티 슬라이드
slidesToShow로 한 번에 여러 슬라이드를 표시합니다. slidesToScroll로 스크롤 단위를 조절합니다.
new Carousel('#container', {
  items: [...],
  slidesToShow: 3,
  slidesToScroll: 1,
  dots: true,
  loop: true
});
카운터 & 썸네일
thumbnails: true로 하단 썸네일 네비게이션, counter: true로 현재/전체 카운터를 표시합니다.
new Carousel('#container', {
  items: [...],
  thumbnails: true,
  counter: true,
  dots: false
});
Pause on Hover
pauseOnHover: true(기본값)로 마우스를 올리면 자동 재생이 일시 중지됩니다. 아래 슬라이더 위에 마우스를 올려보세요.
new Carousel('#container', {
  items: [...],
  autoplay: true,
  autoplaySpeed: 2000,
  pauseOnHover: true  // 기본값
});
옵션 레퍼런스

옵션타입기본값설명

dotsBooleantrue하단 도트 인디케이터
counterBooleanfalse현재/전체 카운터 표시
thumbnailsBooleanfalse썸네일 네비게이션
autoplayBooleanfalse자동 재생
autoplaySpeedNumber3000자동 재생 간격 (ms)
pauseOnHoverBooleantrue호버 시 자동 재생 일시 정지
loopBooleantrue무한 루프
slidesToShowNumber1한 번에 보여줄 슬라이드 수
slidesToScrollNumber1한 번에 스크롤할 슬라이드 수

responsiveArraynull반응형 설정 [{breakpoint, settings}]
onSlideChangeFunctionnull슬라이드 전환 콜백 (current, prev) => {}
onInitFunctionnull초기화 완료 콜백 (carousel) => {}
