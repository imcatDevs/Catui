Tooltips & Popovers

Tooltip과 Popover 컴포넌트 — 요소에 부가 정보를 표시합니다.

Tooltip 방향
placement 옵션으로 4방향 배치가 가능합니다.
const { Tooltip } = await IMCAT.use('tooltips');
new Tooltip('#element', {
  content: '툴팁 텍스트',
  placement: 'top' // 'top' | 'right' | 'bottom' | 'left'
});
버튼 색상별 Tooltip
모든 버튼 스타일에 툴팁을 적용할 수 있습니다.
new Tooltip('#btn', { content: 'Primary 버튼' });
트리거 방식
trigger 옵션으로 hover, click, focus 방식을 지원합니다.
data 속성 방식 
new Tooltip('#el', { content: '클릭!', trigger: 'click' });
new Tooltip('#el', { content: '포커스!', trigger: 'focus' });
// data 속성: <span data-tooltip="텍스트">
HTML 콘텐츠 Tooltip
html: true 옵션으로 풍부한 HTML 콘텐츠를 표시할 수 있습니다.
new Tooltip('#el', {
  content: '<strong>굵은</strong> 텍스트',
  html: true
});
지연 시간 (Delay)
delay 옵션으로 표시/숨김 지연 시간을 조절할 수 있습니다.
new Tooltip('#el', {
  content: '지연 500ms',
  delay: { show: 500, hide: 100 }
});
아이콘 & 인라인 Tooltip
아이콘, 링크, 텍스트 등 다양한 인라인 요소에 적용할 수 있습니다.
help_outline info warning 도움말 링크 HTML 
new Tooltip('#icon', { content: '도움말' });
new Tooltip('#abbr', { content: 'HyperText Markup Language' });
Popover 방향
placement 옵션으로 Popover의 방향을 지정합니다.
const { Popover } = await IMCAT.use('tooltips');
new Popover('#element', {
  title: '제목',
  content: '<p>내용</p>',
  placement: 'top',
  trigger: 'click'
});
Dismissible Popover
외부 클릭 시 자동으로 닫히는 Popover입니다.
new Popover('#el', {
  title: 'Dismissible popover',
  content: '외부를 클릭하면 닫힙니다.',
  dismissible: true
});
Hover Popover & HTML 콘텐츠
trigger: 'hover'로 마우스 오버 시 표시되며, 풍부한 HTML을 담을 수 있습니다.
new Popover('#el', {
  title: 'Hover',
  content: '마우스 오버로 표시',
  trigger: 'hover'
});
