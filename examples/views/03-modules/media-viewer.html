EmbedVideo (JS API)

프로그래밍 방식 임베드: URL 자동 변환 (YouTube/Vimeo), 비율 변경

EmbedVideo — 16:9 + shadow

EmbedVideo — 4:3 + rounded

← 좌측 임베드 비율 변경 
const { EmbedVideo } = await IMCAT.use('media-viewer');

// YouTube URL 자동 변환
const embed = new EmbedVideo('#container', {
  url: 'https://www.youtube.com/watch?v=zpOULjyy-n8',
  ratio: '16x9',
  shadow: true
});

// 비율 변경
embed.setRatio('21x9');

// URL 변경
embed.setSrc('https://vimeo.com/123456789');
VideoPlayer

커스텀 컨트롤: 재생/일시정지, 진행 바, 볼륨, 속도, PIP, 전체 화면

키보드: Space/K 재생 · ←→ 탐색 · ↑↓ 볼륨 · M 음소거 · F 전체 화면

const { VideoPlayer } = await IMCAT.use('media-viewer');

new VideoPlayer('#container', {
  src: 'video.mp4',
  poster: 'thumbnail.jpg',
  controls: true,
  customControls: true,
  autoplay: false,
  playbackRates: [0.5, 1, 1.5, 2],
  onPlay: () => console.log('재생'),
  onEnded: () => console.log('종료')
});
AudioPlayer

커스텀 오디오 플레이어: 앨범 커버, 곡 정보, 진행 바, 볼륨

const { AudioPlayer } = await IMCAT.use('media-viewer');

new AudioPlayer('#container', {
  src: 'audio.mp3',
  title: '음악 제목',
  artist: '아티스트',
  cover: 'cover.jpg',
  volume: 0.8
});

// 트랙 변경
player.setTrack('new.mp3', '새 제목', '새 아티스트');
ImageViewer

이미지 확대/축소, 회전, 다운로드 — 마우스 휠 줌 지원

const { ImageViewer } = await IMCAT.use('media-viewer');

new ImageViewer('#container', {
  src: 'photo.jpg',
  alt: '사진 설명',
  zoomable: true,
  rotatable: true,
  downloadable: true,
  minZoom: 0.5,
  maxZoom: 3
});
API 레퍼런스
클래스주요 옵션주요 메서드
EmbedVideo url, ratio, shadow, rounded, autoplay setSrc(url), setRatio(r), destroy()
VideoPlayer src, poster, customControls, playbackRates play(), pause(), seek(t), togglePIP(), toggleFullscreen()
AudioPlayer src, title, artist, cover, volume play(), pause(), setTrack(src, title, artist, cover)
ImageViewer src, alt, zoomable, rotatable, downloadable zoomIn(), zoomOut(), rotate(deg), reset(), download()
