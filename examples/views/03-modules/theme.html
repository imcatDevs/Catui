Theme

라이트/다크/시스템 테마 — View Transitions API 기반 전환 효과 (circle, fade, slide)

기본 테마 전환

Light / Dark / System 3가지 모드. localStorage 자동 저장, 시스템 테마 변경 감지.

palette — 
const TM = await IMCAT.use('theme');
const theme = new TM.Theme({
  defaultTheme: 'system',  // 'light' | 'dark' | 'system'
  storageKey: 'imcat-theme',
  onChange: (resolved, setting) => {
    console.log('적용:', resolved, '설정:', setting);
  }
});

theme.set('dark');     // 다크 모드
theme.set('system');   // 시스템 모드
theme.toggle();       // light ↔ dark 전환
theme.isDark();       // → true/false
theme.getResolved(); // → 'light' 또는 'dark'
전환 효과

View Transitions API 활용. 미지원 브라우저에서는 폴백 애니메이션 자동 적용.

전환 효과를 선택한 후 토글 버튼을 누르세요:

현재: circle 
const theme = new TM.Theme({
  transition: 'circle',       // 'none' | 'fade' | 'slide' | 'circle' | 'circle-*'
  transitionDuration: 800   // ms
});

// 런타임에 전환 효과 변경
theme.setTransition('fade', 500);

// 전환 효과 타입 상수
Theme.TRANSITIONS.FADE;            // 'fade'
Theme.TRANSITIONS.CIRCLE;          // 'circle'
Theme.TRANSITIONS.CIRCLE_CENTER;   // 'circle-center'
클릭 위치 원형 전환

toggleWithEvent(e) — 마우스 클릭 위치에서 원이 확산되며 테마 전환

touch_app

아무 곳이나 클릭하면 해당 위치에서 원형 전환

// 클릭 이벤트 기반 전환
button.addEventListener('click', (e) => {
  theme.toggleWithEvent(e);
});

// 특정 좌표에서 전환
theme.setWithCircleAt('dark', 100, 200, 800);
커스텀 테마

CSS 변수 기반 커스텀 테마 등록 및 적용

// 커스텀 테마 등록
theme.register('ocean', {
  '--bg-primary': '#0c4a6e',
  '--bg-secondary': '#075985',
  '--text-primary': '#e0f2fe',
  '--border-color': '#0369a1'
});

// 커스텀 테마 적용
theme.set('ocean');
