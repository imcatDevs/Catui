섹션 1 — IMCAT UI 소개

IMCAT UI는 경량 제로빌드 ES6+ JavaScript 웹 프레임워크입니다. 동적 모듈 로딩, SPA 라우팅, XSS 자동 방어를 핵심으로 합니다. 별도의 번들러 설정 없이 브라우저에서 바로 실행 가능하며, 20개 코어 모듈과 22개 확장 UI 컴포넌트를 제공합니다. 모든 컴포넌트는 destroy() 메서드를 통해 메모리 누수를 방지합니다.

섹션 2 — 모듈 시스템

IMCAT.use('모듈명')으로 필요한 모듈을 런타임에 동적 로드합니다. 로드된 모듈은 캐싱되어 반복 요청 시 네트워크 비용이 없습니다. 하위 모듈 추출도 지원하여 IMCAT.use('overlays/modal')처럼 특정 클래스만 가져올 수 있습니다. 각 모듈은 독립적인 CSS를 가지며 autoLoadModuleCSS 옵션으로 자동 로드됩니다.

섹션 3 — 보안

Security 모듈이 XSS 공격 방어, 경로 검증, 입력 새니타이징을 자동으로 처리합니다. 모든 사용자 입력은 Security.escape()로 이스케이프되며 eval()과 new Function() 사용이 프레임워크 전체에서 금지됩니다. HTML 템플릿 엔진에서도 자동 이스케이프가 적용되어 안전한 렌더링을 보장합니다.

섹션 4 — 스타일 시스템

SCSS 변수와 디자인 토큰을 통해 일관된 스타일을 유지합니다. _variables.scss에서 색상, 타이포그래피, 간격, z-index 등을 수정하면 전체 컴포넌트에 자동 반영됩니다. 다크 모드는 theme 모듈을 통해 런타임 전환을 지원하며, BEM 명명 규칙을 일관되게 따릅니다.

const { Scrollspy } = await IMCAT.use('scroll');
new Scrollspy('#nav', {
  scrollContainer: '#scroll-area',  // 스크롤 컨테이너
  targets: 'div[id]',
  offset: 100,
  activeClass: 'is-active',
  smoothScroll: true,
  onChange: (activeId) => console.log(activeId)
});
InfiniteScroll

스크롤이 하단에 도달하면 자동으로 콘텐츠를 로드합니다.

const { InfiniteScroll } = await IMCAT.use('scroll');
new InfiniteScroll('#container', {
  loadMore: async (page) => {
    const res = await fetch('/api/items?page=' + page);
    return res.json();
  },
  threshold: 200,
  pageSize: 20
});
BackToTop
const { BackToTop } = await IMCAT.use('scroll');
new BackToTop({
  threshold: 300,
  smooth: true
});
SmoothScroll

페이지 내 앵커 링크를 부드러운 스크롤로 이동시킵니다.

VirtualScroll 섹션으로 이동 
const { SmoothScroll } = await IMCAT.use('scroll');
// 유틸리티 객체 — new 없이 직접 호출
SmoothScroll.to('#target', { offset: 80, duration: 600 });
SmoothScroll.toTop();   // 맨 위로
SmoothScroll.toBottom(); // 맨 아래로
