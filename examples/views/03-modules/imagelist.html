ImageList

이미지 그리드, 라이트박스, Before/After 비교, 지연 로딩 컴포넌트

ImageList — Grid

CSS Grid 기반 이미지 갤러리. 반응형 columns, 호버 효과(zoom/fade/overlay), 라이트박스 내장.

const { ImageList } = await IMCAT.use('imagelist');
new ImageList('#container', {
  images: [{ src: 'img.jpg', title: '제목' }, ...],
  layout: 'grid',      // grid | masonry | quilted
  columns: 4,
  gap: 8,
  hover: 'zoom',       // zoom | fade | overlay | none
  lightbox: true,
  aspectRatio: '4:3',  // 1:1 | 4:3 | 16:9 | auto
  showTitle: true
});
ImageLightbox

풀스크린 이미지 뷰어. 키보드/스와이프 네비게이션, 줌, 다운로드, 썸네일 스트립 지원.

const { ImageLightbox } = await IMCAT.use('imagelist');
const lb = new ImageLightbox({
  images: [{ src: 'full.jpg', thumb: 'thumb.jpg', title: '캡션' }],
  startIndex: 0,
  thumbnails: true,
  zoom: true,
  download: true,
  keyboard: true,
  swipe: true
});
lb.open();
ImageCompare — Before / After

드래그 슬라이더로 두 이미지를 비교. 수평/수직 모드, 라벨, 핸들 커스터마이징.

수평 모드
수직 모드
const { ImageCompare } = await IMCAT.use('imagelist');
new ImageCompare('#container', {
  before: { src: 'before.jpg', label: 'Before' },
  after: { src: 'after.jpg', label: 'After' },
  orientation: 'horizontal', // horizontal | vertical
  initialPosition: 50
});
LazyImage — 지연 로딩

IntersectionObserver 기반 지연 로딩. blur/skeleton/color 플레이스홀더, 페이드인 애니메이션.

const { LazyImage } = await IMCAT.use('imagelist');
// data-lazy-src 속성이 있는 img 요소를 자동 감지
new LazyImage({
  selector: '[data-lazy-src]',
  placeholder: 'skeleton', // blur | skeleton | color | none
  animation: 'fade',
  rootMargin: '50px',
  onLoad: (img) => console.log('로드 완료', img)
});
