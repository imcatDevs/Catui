Gantt Chart

프로젝트 일정 관리 — 뷰 모드, 의존성, 담당자, 그룹/접기, 진행률, 드래그 편집

프로젝트 간트

뷰 모드 전환(일/주/월), 오늘 라인, 의존성 화살표, 담당자, 그룹 접기, 진행률 표시

const Gantt = await IMCAT.use('gantt');

new Gantt('#container', {
  viewMode: 'day',  // 'day' | 'week' | 'month'
  todayLine: true,
  showProgress: true,
  showDependencies: true,
  tasks: [
    {
      id: 'design', name: '디자인',
      start: '2025-07-01', end: '2025-07-10',
      progress: 100, assignee: '김영희',
      color: '#8b5cf6',
      children: [
        { id: 'wire', name: '와이어프레임', ... },
        { id: 'ui', name: 'UI 디자인', dependencies: ['wire'], ... }
      ]
    },
    {
      id: 'dev', name: '개발',
      dependencies: ['design'], ...
    }
  ],
  onTaskClick: (task) => console.log(task)
});
편집 가능 간트

드래그로 작업 이동/크기 조절. 바 좌우 핸들로 시작/종료일 변경.

new Gantt('#container', {
  editable: true,
  tasks: [...],
  onTaskChange: (task) => {
    console.log('변경:', task.name, task.start, task.end);
  }
});
