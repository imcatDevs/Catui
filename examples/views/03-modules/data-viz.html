Data Visualization

DataTable, Chart, Kanban, Calendar, Masonry — 데이터 시각화 컴포넌트 모음

DataTable

정렬, 검색, 페이지네이션, 행 선택, CSV 내보내기, 커스텀 렌더러

const { DataTable } = await IMCAT.use('data-viz');

new DataTable('#container', {
  columns: [
    { field: 'name', title: '이름', sortable: true },
    { field: 'role', title: '직책' },
    { field: 'status', title: '상태',
      render: (v) => `<span class="badge">${v}</span>` }
  ],
  data: [...],
  paginate: true,
  pageSize: 5,
  searchable: true,
  selectable: true,
  exportable: true,
  striped: true
});

Kanban

드래그 앤 드롭, 우선순위, 태그, 담당자, WIP 제한, 체크박스

new Kanban('#container', {
  columns: [
    {
      id: 'todo', title: '할 일', color: '#667eea',
      cards: [{
        id: '1', title: 'UI 디자인',
        priority: 'high',
        tags: [{ text: '디자인', color: '#8b5cf6' }],
        assignees: [{ name: '홍길동' }],
        date: '3/20'
      }]
    },
    { id: 'doing', title: '진행 중', color: '#f59e0b', wipLimit: 3, cards: [] },
    { id: 'done', title: '완료', color: '#22c55e', cards: [] }
  ],
  onMove: (cardId, from, to) => console.log('이동:', cardId, from, '→', to)
});
Calendar

월간/주간 뷰 전환, 오늘 버튼, 이벤트 카테고리, 날짜/이벤트 클릭

new Calendar('#container', {
  view: 'month',  // 'month' | 'week'
  events: [
    { id: '1', date: '2025-03-20', title: '회의', category: 'primary' },
    { id: '2', date: '2025-03-25', title: '마감', category: 'danger' }
  ],
  maxEventsPerCell: 2,
  onDateClick: (date) => console.log(date),
  onEventClick: (id) => console.log(id)
});

// API
cal.addEvent({ id: '3', date: '2025-04-01', title: '출시' });
cal.setView('week');
cal.goToToday();
Masonry

CSS Grid 기반 타일 레이아웃, 필터, 애니메이션

new Masonry('#container', {
  columnWidth: 280,
  gap: 16,
  animate: true,
  filterKey: 'category',
  items: [...],
  render: (item) => `<div class="card">...</div>`
});

masonry.filter('디자인');   // 카테고리 필터
masonry.clearFilter();      // 필터 해제
