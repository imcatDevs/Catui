File Uploads

드래그 앤 드롭 파일 업로드. 미리보기, 다중 업로드, 파일 타입 필터 지원.

Dropzone

드래그 앤 드롭 영역 + 클릭으로 파일 업로드. 미리보기 자동 표시.

Basic Dropzone

new FileUpload('#el') — 기본 드롭존. 모든 파일 타입 허용.

Image Only

accept: 'image/*' — 이미지 파일만 허용. 미리보기 썸네일 표시.

Multiple Files

multiple: true, maxFiles: 5 — 최대 5개 파일 동시 업로드.

Size Limit

maxSize: 2 * 1024 * 1024 — 2MB 파일 크기 제한.

Accept PDF + Images

accept: 'image/*,.pdf' — 이미지와 PDF만 허용.

onChange Callback

onChange: (files) => { ... } — 파일 변경 시 콜백.

파일 없음

const F = await IMCAT.use('forms');
new F.FileUpload('#el', {
  accept: 'image/*,.pdf',
  multiple: true,
  maxSize: 5 * 1024 * 1024,
  maxFiles: 10,
  dropzoneText: '파일을 드래그하세요',
  onChange: (files) => console.log(files)
});
