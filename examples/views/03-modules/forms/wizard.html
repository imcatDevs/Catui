Form Wizard

단계별 폼 위저드. 진행률 표시, 유효성 검사, 완료 콜백 지원.

Basic Wizard

5단계 학생 등록 폼. 이전/다음 버튼으로 탐색. 각 단계별 폼 필드 구성.

const F = await IMCAT.use('forms');
new F.FormWizard('#el', {
  steps: [
    { title: '학생 정보', content: '...' },
    { title: '주소', content: '...' },
    ...
  ],
  onComplete: (data) => console.log(data)
});
Validation Support

각 단계에 validate 콜백 설정. 검증 실패 시 다음 단계 진행 차단.

Without Progress Bar

showProgress: false — 단계 인디케이터 없이 콘텐츠와 네비게이션만 표시.

onComplete Callback

마지막 단계에서 완료 시 모든 패널의 폼 데이터를 자동 수집하여 콜백 전달.
