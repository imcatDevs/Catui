Pickers

날짜, 시간, 색상 선택 및 Countdown/D-Day 컴포넌트

DatePicker & TimePicker

캘린더 기반 날짜 선택과 드롭다운 시간 선택

Datepicker
Basic

new DatePicker('#el') — 기본 날짜 선택.

Default Date

picker.setValue('2026-01-15') — 초기값 설정.

onChange Callback

onChange: (date) => { ... } — 날짜 변경 시 콜백. 아래 선택 결과가 표시됩니다.

아직 선택되지 않음

Timepicker
Basic (15분 간격)

new TimePicker('#el', { step: 15 }) — 15분 단위.

30분 간격

new TimePicker('#el', { step: 30 }) — 30분 단위.

Preloading Time

picker.setValue('14:30') — 초기 시간 설정.

const P = await IMCAT.use('pickers');

// DatePicker
new P.DatePicker('#input', {
  format: 'YYYY-MM-DD',
  onChange: (date) => console.log(date)
});

// TimePicker
new P.TimePicker('#input', {
  step: 15,
  onChange: (time) => console.log(time)
});
ColorPicker

프리셋 색상 + 네이티브 컬러 피커. 색상 미리보기 지원.

기본 프리셋

new ColorPicker('#el') — 기본 12색 프리셋 + 네이티브 컬러 인풋.

커스텀 프리셋

presetColors: [...] — 프리셋 색상을 커스터마이징.

onChange Callback

onChange: (color) => { ... } — 실시간 색상 미리보기.

#10B981 
const { ColorPicker } = await IMCAT.use('pickers');
new ColorPicker('#input', {
  defaultColor: '#3B82F6',
  presetColors: ['#EF4444', '#22C55E', '#3B82F6'],
  onChange: (color) => console.log(color)
});
Countdown & D-Day

실시간 카운트다운 타이머와 D-Day 카운터

Countdown
24시간 카운트다운

targetDate, showLabels: true — 일/시/분/초 표시.

1시간 (일 숨김)

showDays: false — 일 단위를 숨기고 시간에 합산.

라벨 없는 카운트다운

showLabels: false — 숫자만 표시.

D-Day Counter
미래 (D-N)

targetDate가 미래인 경우 — D-100 등으로 표시.

오늘 (D-Day)

targetDate가 오늘인 경우 — "D-Day" 표시.

과거 (D+N)

showPastDays: true — 지난 일수 표시.

const P = await IMCAT.use('pickers');

// Countdown
new P.Countdown('#el', {
  targetDate: new Date(Date.now() + 86400000),
  showLabels: true,
  onComplete: () => alert('완료!')
});

// DDay
new P.DDay('#el', {
  targetDate: new Date('2026-12-31'),
  title: '프로젝트 마감',
  showPastDays: true
});
