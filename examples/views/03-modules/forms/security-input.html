Security Input

OTP 인증 코드, PIN 보안 입력. 자동 포커스 이동, 붙여넣기, 키패드 지원.

Two-Factor Authentication

6자리 인증 코드 입력. 자동 포커스 이동 + 붙여넣기 지원.

인증 코드가 ******6789 으로 전송되었습니다.

코드를 못 받으셨나요? 재전송 

const SI = await IMCAT.use('security-input');
new SI.OTPInput('#container', {
  length: 6,
  type: 'number',
  onComplete: (code) => console.log(code)
});
OTP 변형

구분선, 텍스트 타입, 마스킹, 크기 변형 등 다양한 OTP 옵션

구분선 (Separator)

separator: true, separatorPosition: 3 — 3-3 형태 구분

마스킹 (Masked)

mask: true — 입력값을 ●로 마스킹

4자리 텍스트

length: 4, type: 'text' — 영문+숫자 혼합 코드

Small / Default / Large

PIN 변형

6자리, 마스킹 해제, 에러/성공 상태

6자리 PIN

length: 6 — 6자리 보안 PIN

마스킹 해제

mask: false — 입력값 그대로 표시

에러 / 성공 상태

올바른 PIN: 1234 → 성공, 그 외 → 에러

가상 키패드

keypad: true — 화면 키패드로 입력. 모바일/키로거 방지.

셔플 키패드

shuffleKeypad: true — 매번 키 배치 변경으로 보안 강화. 완료 시 자동 셔플.
