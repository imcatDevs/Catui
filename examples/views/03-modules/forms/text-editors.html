Text Editors

리치 텍스트 에디터, 마크다운 에디터, 자동 높이 조절 Textarea

RichTextEditor

간단한 WYSIWYG 에디터. 툴바 기반 서식 지정, 키보드 단축키(Ctrl+B/I/U) 지원.

const { RichTextEditor } = await IMCAT.use('text-editors');
const editor = new RichTextEditor('#container', {
  toolbar: ['bold', 'italic', 'underline', 'strikethrough', '|',
    'heading', 'quote', '|', 'ul', 'ol', '|',
    'link', 'image', '|', 'code', 'hr', '|',
    'alignLeft', 'alignCenter', 'alignRight', '|',
    'undo', 'redo'],
  placeholder: '내용을 입력하세요...',
  minHeight: 200,
  onChange: (html) => console.log(html)
});
editor.getHTML();  // HTML 반환
editor.getText();  // 텍스트 반환
editor.clear();    // 초기화
MarkdownEditor — 분할 뷰

마크다운 에디터 + 실시간 프리뷰. 좌측 편집, 우측 미리보기 분할 레이아웃.

const { MarkdownEditor } = await IMCAT.use('text-editors');
new MarkdownEditor('#container', {
  splitView: true,   // 분할 뷰
  preview: true,
  minHeight: 300,
  value: '# 제목\n\n본문 텍스트',
  onChange: (md) => console.log(md)
});
MarkdownEditor — 탭 모드

편집/미리보기 탭 전환 모드. 좁은 화면에서 유용합니다.

new MarkdownEditor('#container', {
  splitView: false,  // 탭 전환 모드
  preview: true,
  minHeight: 250,
  toolbar: ['bold', 'italic', '|', 'h1', 'h2', 'h3', '|',
    'ul', 'ol', 'task', '|', 'link', 'code', 'codeblock', '|',
    'quote', 'table']
});
TextareaAutosize — 기본

기존 <textarea>에 적용하여 내용에 따라 자동으로 높이 조절. minRows/maxRows 제한 가능.

const { TextareaAutosize } = await IMCAT.use('text-editors');
// 기존 textarea 요소에 적용
new TextareaAutosize('#my-textarea', {
  minRows: 3,
  maxRows: 10
});
TextareaAutosize — 툴바 + 글자 수

선택적 도구 모음(초기화, 복사, 실행 취소/다시 실행)과 하단 글자 수/줄 수 상태바.

new TextareaAutosize('#my-textarea', {
  minRows: 3,
  maxRows: 15,
  toolbar: ['undo', 'redo', '|', 'copy', 'clear'],
  showCount: true,
  maxLength: 500,
  onChange: (value) => console.log(value)
});
