Range Slider

드래그 가능한 범위 슬라이더. 단일/이중 핸들, 툴팁, 커스텀 포맷 지원.

Range Slider Examples

다양한 RangeSlider 변형 데모

Default Range

range: true, value: [20, 80] — 기본 이중 핸들 범위.

Single Handle

range: false, value: 50 — 단일 핸들 슬라이더.

Step: 10

step: 10 — 10 단위로 스냅.

Step: 25

step: 25 — 25 단위 스냅 (0, 25, 50, 75, 100).

Currency Format

formatValue: (v) => v.toLocaleString() + '원'

Percent Format

formatValue: (v) => v + '%'

Without Tooltip

showTooltip: false — 핸들 위 툴팁 숨김.

Without Labels

showLabels: false — 양쪽 라벨 숨김.

onChange Callback

onChange — 실시간 값 표시.

값: [25, 75]

const S = await IMCAT.use('selectors');
new S.RangeSlider('#el', {
  min: 0, max: 100,
  value: [20, 80],
  step: 1,
  range: true,
  showTooltip: true,
  showLabels: true,
  formatValue: (v) => v + '%',
  onChange: (val) => console.log(val)
});
