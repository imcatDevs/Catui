Selectors

Autocomplete, MultiSelect, RangeSlider 컴포넌트

Autocomplete

텍스트 입력 시 자동완성 드롭다운. 배열 또는 async 함수 소스 지원.

Basic

new Autocomplete('#el', { source: [...] }) — 배열 기반 자동완성.

Highlight Match

highlight: true — 검색어 일치 부분 강조 (기본값).

Async Source

source: async (query) => [...] — 비동기 함수 소스. 딜레이 시뮬레이션.

onSelect Callback

onSelect: (item) => { ... } — 항목 선택 시 콜백.

아직 선택되지 않음

const S = await IMCAT.use('selectors');
new S.Autocomplete('#input', {
  source: ['서울', '부산', '대구'],
  minLength: 1,
  maxResults: 10,
  highlight: true,
  onSelect: (item) => console.log(item)
});
MultiSelect

태그 기반 다중 선택. 검색, 태그 제거, 커스텀 태그 생성 지원.

Default

new MultiSelect('#el', { options: [...] }) — 기본 다중 선택.

Pre-selected + Remove

selected: ['js','ts'] — 초기 선택값. × 버튼으로 개별 제거.

Allow Create

allowCreate: true — 목록에 없는 값도 직접 입력하여 추가.

Max Items Limit

maxItems: 3 — 최대 선택 수 제한.

onChange Callback

onChange: (values) => { ... } — 선택 변경 시 콜백.

선택: []

const S = await IMCAT.use('selectors');
new S.MultiSelect('#el', {
  options: [
    { value: 'js', label: 'JavaScript' },
    { value: 'ts', label: 'TypeScript' }
  ],
  selected: ['js'],
  allowCreate: true,
  maxItems: 3,
  onChange: (values) => console.log(values)
});
RangeSlider

드래그 가능한 범위 슬라이더. 단일/이중 핸들, 툴팁, 커스텀 포맷 지원.

Range (이중 핸들)

range: true, value: [20, 80] — 기본 범위 선택.

Single (단일 핸들)

range: false, value: 50 — 단일 값 선택.

Custom Format

formatValue: (v) => v.toLocaleString() + '원' — 값 포맷 커스터마이징.

Step

step: 10 — 10 단위로 이동.

onChange Callback

onChange: (value) => { ... } — 드래그 중 실시간 값 표시.

값: [25, 75]

const S = await IMCAT.use('selectors');
new S.RangeSlider('#el', {
  min: 0, max: 100,
  value: [20, 80],
  step: 1,
  range: true,
  showTooltip: true,
  formatValue: (v) => v + '%',
  onChange: (val) => console.log(val)
});
