Other Plugins

Rating(별점), SignaturePad(서명) 등 폼 확장 컴포넌트

Rating

별점 평가 컴포넌트. 클릭/호버로 점수 선택.

Basic (5점)

new Rating('#el', { max: 5, value: 3 })

10점 만점

max: 10, value: 7 — 10점 스케일.

Readonly

readonly: true — 읽기 전용 표시.

onChange Callback

onChange: (value) => { ... } — 선택 시 콜백.

선택: 0점

const F = await IMCAT.use('forms');
new F.Rating('#el', {
  max: 5,
  value: 3,
  onChange: (v) => console.log(v)
});
SignaturePad

마우스/터치 서명 패드. 지우기, 저장(PNG) 지원.

Basic

new SignaturePad('#el') — 기본 서명 패드. 지우기/저장 버튼 포함.

Custom Size & Pen Color

width: 300, penColor: '#2563EB', penWidth: 3

Background Color

backgroundColor: '#F0FDF4' — 배경색 커스터마이징.

const F = await IMCAT.use('forms');
const pad = new F.SignaturePad('#el', {
  width: 400,
  height: 200,
  penColor: '#000',
  penWidth: 2
});
pad.toDataURL();  // base64 이미지
pad.clear();       // 지우기
