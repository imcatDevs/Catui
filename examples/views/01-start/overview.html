빠른 시작

IMCAT UI를 프로젝트에 추가하고 첫 번째 컴포넌트를 사용해보세요.

CDN으로 시작하기
HTML 파일에 CSS와 JS를 추가하는 것만으로 바로 사용할 수 있습니다.
<!-- 1. CSS -->
<link rel="stylesheet" href="dist/imcat-ui.css">

<!-- 2. JS (body 끝) -->
<script src="dist/imcat-ui.min.js"></script>

<!-- 3. 사용 -->
<script>
  IMCAT.toast.success('IMCAT UI 로드 완료!');
</script>
ES Module
번들러(Vite, Webpack 등) 환경에서는 ESM으로 임포트합니다.
import IMCAT from './dist/imcat-ui.esm.js';

const { Modal } = await IMCAT.use('overlays');
new Modal({ title: 'Hello', content: 'World' }).show();
단축 API
모듈을 직접 로드하지 않아도 자주 쓰는 기능을 한 줄로 호출할 수 있습니다.
// Alert
await IMCAT.alert('저장되었습니다');

// Confirm
if (await IMCAT.confirm('삭제하시겠습니까?')) { ... }

// Prompt
const name = await IMCAT.prompt('이름을 입력하세요');

// Toast
IMCAT.toast.success('완료!');
IMCAT.toast.error('오류 발생');
IMCAT.toast.warning('주의');
IMCAT.toast.info('안내');
동적 모듈 로드
IMCAT.use()로 필요한 모듈만 런타임에 로드합니다. CSS도 자동으로 함께 로드됩니다.
// 전체 모듈 로드
const Overlays = await IMCAT.use('overlays');
new Overlays.Modal({ title: '제목' }).show();

// 하위 모듈만 로드
const Modal = await IMCAT.use('overlays/modal');
new Modal({ title: '제목' }).show();

// 여러 모듈 동시 로드
const [Nav, Pickers] = await Promise.all([
  IMCAT.use('navigation'),
  IMCAT.use('pickers')
]);
SPA 라우팅
catui-href 속성으로 페이지 새로고침 없이 뷰를 전환합니다.
<!-- catui-target: 뷰가 렌더링될 컨테이너 -->
<body catui-target="content">
  <a catui-href="views/home.html">홈</a>
  <a catui-href="views/about.html">소개</a>
  <div id="content"></div>
</body>

// JS로 네비게이션
IMCAT.view.navigate('views/dashboard.html');

// 라이프사이클 훅
IMCAT.view.afterLoad((path) => {
  console.log('로드됨:', path);
});
