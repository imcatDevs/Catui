State Management

반응형 상태 관리. 상태 변경 시 자동으로 구독자에게 알립니다.

기본 사용
카운트: 0
// 상태 생성 (리액티브 Proxy 반환)
const store = IMCAT.state.create({ count: 0 });

// 특정 키 변경 감지
store.watch('count', (newVal, oldVal) => {
  console.log('변경됨:', oldVal, '->', newVal);
});

// 상태 변경 (직접 할당 — 자동 감지)
store.count++;

// 현재 상태 조회
const current = store.getState(); // { count: 1 }
파생 상태 (compute)
₩30,000
// 파생 상태 — 의존성 자동 추적 (Proxy)
store.compute('total', function() {
  return this.price * this.qty;
});
일괄 변경 (batch / setState)
사용자: -
// batch — 복수 변경을 1번만 통지
store.batch(function() {
  store.name = '홍길동';
  store.role = 'Admin';
});

// setState — 부분 업데이트
store.setState({ name: '김영희', role: 'Editor' });
API 레퍼런스
IMCAT.state.create(initialState)    // 리액티브 Proxy 생성
store.watch(key, callback)          // 키 변경 감지
store.unwatch(key, callback)        // 감지 해제
store.getState()                    // 전체 상태 스냅샷
store.setState(partial)             // 상태 일괄 업데이트
store.reset()                       // 초기값으로 리셋
store.compute(key, getter)          // 파생 상태 (의존성 자동 추적)
store.batch(fn)                     // 일괄 변경
