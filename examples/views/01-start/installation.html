설치

CDN, npm, 직접 다운로드 세 가지 방법으로 IMCAT UI를 설치할 수 있습니다.

CDN (권장)
별도 설치 없이 HTML에 바로 추가합니다.
<head>
  <link rel="stylesheet" href="dist/imcat-ui.css">
</head>
<body>
  <!-- 콘텐츠 -->
  <script src="dist/imcat-ui.min.js"></script>
</body>
npm / pnpm
패키지 매니저를 통한 설치입니다.
# npm
npm install @imcat-ckim/catui

# pnpm
pnpm add @imcat-ckim/catui
ES Module import
번들러 환경에서 ESM으로 사용합니다.
import IMCAT from '@imcat-ckim/catui/dist/imcat-ui.esm.js';

// 또는 IIFE 번들이 이미 로드된 경우 전역 IMCAT 사용
const { Modal } = await IMCAT.use('overlays');
빌드 산출물 구조
dist/ 폴더에 포함된 파일들입니다.
dist/
├── imcat-ui.css          // 코어 CSS (타이포, 버튼, 카드 등)
├── imcat-ui.js           // IIFE 번들 (전역 IMCAT)
├── imcat-ui.min.js       // 압축 IIFE
├── imcat-ui.esm.js       // ES Module 번들
└── modules/
    ├── overlays.js       // 개별 모듈 JS
    ├── overlays.css      // 개별 모듈 CSS
    ├── navigation.js
    ├── navigation.css
    └── ...               // 22개 모듈
