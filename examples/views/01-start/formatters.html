Formatters

숫자, 날짜, 바이트, 전화번호 등 다양한 포맷터를 제공합니다.

숫자 포맷
입력결과
 
IMCAT.format.number(1234567);       // '1,234,567'
IMCAT.format.currency(49900);       // '₩49,900'
IMCAT.format.percent(0.856);        // '85.6%'
IMCAT.format.bytes(1536000);        // '1.46 MB'
날짜 포맷
함수결과
 
IMCAT.format.date(new Date());         // '2025-03-14'
IMCAT.format.time(new Date());         // '14:30:00'
IMCAT.format.datetime(new Date());     // '2025-03-14 14:30:00'
IMCAT.format.relative(new Date());     // '방금 전'
텍스트 포맷
IMCAT.format.phone('01012345678');     // '010-1234-5678'
IMCAT.format.mask('홍길동', 1);        // '홍**'
IMCAT.format.truncate('긴 텍스트...', 10); // '긴 텍스트...'
