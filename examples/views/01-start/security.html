Security

XSS 방어, HTML 새니타이징, 경로 검증 등 보안 유틸리티를 제공합니다.

XSS 이스케이프
// HTML 특수문자 이스케이프
IMCAT.security.escape('<script>alert("xss")</script>');
// → '&lt;script&gt;alert(&quot;xss&quot;)&lt;/script&gt;'

// 단축 호출
IMCAT.escape(userInput);
HTML 새니타이징
// 위험한 태그/속성 제거, 안전한 HTML만 유지
IMCAT.security.sanitize(
  '<p>안전</p><script>위험</script><img onerror="hack()">'
);
// → '<p>안전</p>'
경로 & URL 검증
함수입력결과
 
IMCAT.security.validatePath('views/home.html');   // true
IMCAT.security.validatePath('../../etc/passwd');   // false
IMCAT.security.isSafeUrl('https://example.com');  // true
IMCAT.security.isSafeUrl('javascript:alert(1)');  // false
IMCAT.security.isSafeFilename('doc.pdf');          // true
IMCAT.security.sanitizeCSS('expression(evil)');   // ''
보안 규칙 요약
필수 규칙
  • innerHTML에 사용자 입력 삽입 시 반드시 Security.escape() 처리
  • textContent 사용 시에는 이스케이프 불필요 (브라우저가 자동 처리)
  • eval(), new Function() 사용 절대 금지
  • URL 파라미터 값은 이스케이프되지 않은 원본 — DOM 삽입 시 주의
