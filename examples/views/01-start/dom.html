DOM API

jQuery와 유사한 체이닝 DOM 조작 API. IMCAT(selector)로 호출합니다.

요소 선택
이 요소를 선택합니다
IMCAT('#element')         // ID 선택
IMCAT('.class')           // 클래스 선택
IMCAT('div.card')         // CSS 셀렉터
IMCAT(domElement)          // DOM 요소 래핑
클래스 조작
클릭하여 토글
IMCAT('.el').addClass('active');
IMCAT('.el').removeClass('active');
IMCAT('.el').toggleClass('active');
IMCAT('.el').hasClass('active'); // boolean
콘텐츠 조작
원본 텍스트
IMCAT('.el').text('새 텍스트');
IMCAT('.el').html('<b>HTML</b>');
IMCAT('.el').attr('title', '새 타이틀');
IMCAT('.el').css('color', 'red');
