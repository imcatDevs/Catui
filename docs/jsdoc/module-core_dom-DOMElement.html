DOMElement

core/dom~ DOMElement

DOM Element Wrapper

Constructor

new DOMElement(elements)

Description:

  • jQuery 스타일의 체이닝 가능한 DOM 조작 API를 제공하는 래퍼 클래스입니다. 하나 또는 여러 개의 DOM 요소를 감싸서 편리한 메서드 체이닝을 제공합니다.

Source:
  • dom.js, line 21
Example
// jQuery 스타일 체이닝
new DOMElement(element)
  .addClass('active')
  .text('Hello')
  .on('click', handler);
