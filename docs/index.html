Core API

문서설명
DOMjQuery 스타일 DOM 조작, 체이닝, 자동 XSS 방어
RouterSPA 라우팅, catui-href 속성, 뷰 관리
EventsEventBus (글로벌) + EventEmitterMixin (컴포넌트)
StateProxy 기반 리액티브 상태 관리
APIHTTP 클라이언트 (fetch 래퍼, 인터셉터)
Config글로벌 설정, 컴포넌트별 기본값
Formatters숫자, 날짜, 통화, 전화번호 포맷
Template머스태시 템플릿 엔진, 자동 이스케이프
StoragelocalStorage/sessionStorage + TTL
